#### Installation

Copy-paste this into your terminal:

```sh
git clone https://github.com/mkdirlove/WP-Scanner.git
```
```
cd WP-Scanner
```
```
python3 -m pip install -r requirements.txt
```
```
python3 WP-Scanner.py -h
```
or
```
python3 WP_ScannerV2.py -h
```
